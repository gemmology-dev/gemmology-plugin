# Physical Properties of Gemstones

Guidance on understanding and measuring the physical properties of coloured gemstones, essential for identification and practical applications in jewellery.

## Core Physical Properties

### Hardness (Mohs Scale)

Hardness measures resistance to scratching. Use the Mohs scale (1-10) as a relative comparison.

| Mohs | Reference Mineral | Common Gemstones |
|------|-------------------|------------------|
| 10 | Diamond | - |
| 9 | Corundum | Ruby, Sapphire |
| 8.5 | - | Chrysoberyl, Alexandrite |
| 8 | Topaz | Topaz, Spinel |
| 7.5 | - | Beryl (Emerald, Aquamarine), Zircon |
| 7 | Quartz | Tourmaline, Garnet, Amethyst |
| 6.5 | - | Peridot, Tanzanite, Spodumene |
| 6 | Feldspar | Moonstone, Labradorite, Opal |
| 5.5 | - | Lapis Lazuli |
| 5 | Apatite | Apatite |

**Practical applications**:
- Stones ≥7 suitable for everyday ring wear
- Stones 6-7 require protective settings
- Stones <6 better for earrings, pendants, brooches

**Testing notes**:
- Never scratch-test valuable stones
- Hardness can vary by crystal direction (differential hardness)
- Kyanite: 4-4.5 parallel to length, 6-7 perpendicular

#### Directional Hardness

Some minerals display significantly different hardness depending on crystal direction:

| Mineral | Direction 1 | Direction 2 | Significance |
|---------|-------------|-------------|--------------|
| Kyanite | 4.5 (parallel to c-axis) | 6.5 (perpendicular to c-axis) | Diagnostic identification property |
| Diamond | Variable by face | Hardest on octahedral face, softest on dodecahedral | Critical for cutting and polishing direction |
| Calcite | 3 (on rhombohedron) | 2.5 (on basal pinacoid) | Minor variation |

### Crystal Habit

External form of a crystal as commonly found in nature.

| Habit | Description | Examples |
|-------|-------------|----------|
| Prismatic | Elongated along one axis | Tourmaline, Beryl, Quartz |
| Tabular | Flat, plate-like | Feldspar, Barite, Wulfenite |
| Acicular | Needle-like, very thin and elongated | Rutile (silk in corundum), Natrolite |
| Massive | No visible crystal form | Nephrite jade, Chalcedony, Turquoise |
| Botryoidal | Grape-like rounded surfaces | Malachite, Hematite, Chalcedony |
| Fibrous | Composed of fine parallel fibres | Tiger's eye, Satin spar gypsum |
| Columnar | Thick prismatic crystals | Corundum, Beryl |
| Rhombohedral | Rhombohedron-shaped | Calcite, Dolomite |
| Octahedral | Eight-faced form | Diamond, Spinel, Fluorite |
| Dodecahedral | Twelve-faced form | Garnet |
| Stubby/Short prismatic | Short, blocky crystals | Zircon, Apatite |
| Barrel-shaped | Rounded prism tapering at ends | Corundum (ruby, sapphire) |

### Specific Gravity (SG)

Specific gravity is the ratio of a gemstone's weight to the weight of an equal volume of water.

#### Hydrostatic Weighing (Archimedes Method)

The most accurate method for loose stones. Requires a balance readable to 0.01 ct.

**Procedure**:
1. Weigh stone in air (W_air)
2. Suspend stone in distilled water at 20 °C using a fine wire basket or bridge
3. Weigh stone in water (W_water)
4. Calculate: **SG = W_air / (W_air − W_water)**

**Error sources**:
- Air bubbles clinging to stone surface → brush stone underwater
- Water temperature variation (density changes ~0.02% per °C)
- Surface tension on suspension wire → use thin wire, tare correctly
- Stone touching container walls or base
- Absorbed water in porous stones (opal, turquoise) → use toluene instead

**Worked example**:
- W_air = 5.20 ct, W_water = 3.89 ct
- SG = 5.20 / (5.20 − 3.89) = 5.20 / 1.31 = **3.97** (consistent with corundum)

#### Heavy Liquid Methods

Quick sorting method using liquids of known density. **Safety**: many heavy liquids are toxic; use with proper ventilation and gloves.

| Liquid | SG | Use |
|--------|------|-----|
| Methylene iodide (diiodomethane) | 3.32 | Separates most gems; dilutable with toluene |
| Bromoform (tribromomethane) | 2.89 | Separates quartz (2.65) from feldspar (2.56) |
| Clerici solution | Up to 4.28 | Rarely used due to extreme toxicity (thallium) |

**Interpretation**: Stone sinks (SG > liquid), floats (SG < liquid), or suspends (SG ≈ liquid).

#### SG Diagnostic Separation Tables

| SG Range | Species |
|----------|---------|
| < 2.00 | Amber (1.08), Opal (2.00) |
| 2.00–2.50 | Opal (2.10), Sodalite (2.15–2.35) |
| 2.50–2.80 | Quartz (2.65), Feldspar (2.56–2.76), Beryl (2.72), Turquoise (2.60–2.80), Scapolite (2.57–2.74) |
| 2.80–3.20 | Tourmaline (3.06), Apatite (3.18), Fluorite (3.18), Nephrite (2.95), Jadeite (3.34) |
| 3.20–3.60 | Jadeite (3.34), Spinel (3.60), Diamond (3.52), Topaz (3.53), Peridot (3.34), Sphene (3.53) |
| 3.60–4.20 | Spinel (3.60), Garnet group (3.50–4.20), Corundum (4.00), Chrysoberyl (3.73), Zircon (4.00–4.70) |
| > 4.20 | Zircon (4.00–4.70), Scheelite (6.10), Cassiterite (6.95) |

### Crystal Systems

Seven crystal systems determine optical behaviour and physical properties.

| System | Symmetry | Optical Character | Examples |
|--------|----------|-------------------|----------|
| Cubic (Isometric) | Highest | Singly refractive | Spinel, Garnet, Diamond |
| Tetragonal | 4-fold axis | Doubly refractive (uniaxial) | Zircon |
| Hexagonal | 6-fold axis | Doubly refractive (uniaxial) | Beryl, Apatite |
| Trigonal | 3-fold axis | Doubly refractive (uniaxial) | Corundum, Tourmaline, Quartz |
| Orthorhombic | 3 unequal axes | Doubly refractive (biaxial) | Topaz, Peridot, Tanzanite |
| Monoclinic | 1 oblique axis | Doubly refractive (biaxial) | Spodumene, Orthoclase |
| Triclinic | No symmetry | Doubly refractive (biaxial) | Labradorite, Turquoise |

### Cleavage and Fracture

**Cleavage**: Tendency to break along crystal planes.

| Quality | Description | Examples |
|---------|-------------|----------|
| Perfect | Smooth, flat surfaces | Topaz (basal), Kunzite, Fluorite |
| Good | Distinct but less smooth | Spodumene, Feldspar |
| Poor/Indistinct | Rarely seen | Beryl, Corundum |
| None | No cleavage planes | Quartz, Garnet, Spinel |

**Fracture types**:
- **Conchoidal**: Shell-like curves (quartz, obsidian, glass)
- **Uneven/Irregular**: Rough surface (most gems)
- **Splintery**: Fibrous appearance (nephrite jade)
- **Hackly**: Jagged, sharp edges (native metals)

**Setting considerations**:
- Perfect cleavage stones (topaz, kunzite) require careful setting
- Avoid ultrasonic cleaners for stones with cleavage
- Bezel settings offer more protection than prongs

### Lustre

Surface appearance when light reflects.

| Lustre Type | Description | Examples |
|-------------|-------------|----------|
| Adamantine | Brilliant, diamond-like | Zircon, Demantoid, Sphene |
| Vitreous | Glass-like (most common) | Corundum, Beryl, Quartz, Tourmaline |
| Resinous | Resin-like | Amber, some Garnets |
| Waxy | Wax-like | Turquoise, Jadeite |
| Silky | Fibrous sheen | Tiger's Eye, Satin Spar |
| Pearly | Pearl-like | Moonstone |
| Greasy | Oily appearance | Nephrite, some Serpentine |

### Tenacity

Resistance to breaking, crushing, bending.

| Type | Description | Examples |
|------|-------------|----------|
| Brittle | Breaks or powders easily | Most gemstones |
| Tough | Resists breaking | Nephrite jade, Jadeite |
| Sectile | Can be cut with knife | Amber |

**Toughness vs hardness**:
- Nephrite (H 6-6.5) is tougher than diamond (H 10)
- Jadeite exceptionally tough despite moderate hardness
- Important for carving and durability
