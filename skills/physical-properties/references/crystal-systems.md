# Crystal Systems Reference

The seven crystal systems and their gemmological significance.

## Overview

| System | Axes | Angles | Optical Character | Examples |
|--------|------|--------|-------------------|----------|
| **Cubic** | a = b = c | All 90° | Isotropic (SR) | Spinel, Garnet, Diamond |
| **Tetragonal** | a = b ≠ c | All 90° | Uniaxial | Zircon, Scapolite |
| **Hexagonal** | a₁ = a₂ = a₃ ≠ c | 60°/90° | Uniaxial | Beryl, Apatite |
| **Trigonal** | a₁ = a₂ = a₃ ≠ c | 60°/90° | Uniaxial | Corundum, Tourmaline, Quartz |
| **Orthorhombic** | a ≠ b ≠ c | All 90° | Biaxial | Topaz, Peridot, Tanzanite |
| **Monoclinic** | a ≠ b ≠ c | Two 90°, one ≠90° | Biaxial | Spodumene, Orthoclase, Jadeite |
| **Triclinic** | a ≠ b ≠ c | None 90° | Biaxial | Labradorite, Turquoise |

## Cubic (Isometric) System

**Characteristics**:
- Highest symmetry
- Three equal axes at 90°
- Singly refractive (isotropic)
- No pleochroism
- No birefringence

**Gem Examples**:
| Gemstone | Crystal Habit | Notes |
|----------|---------------|-------|
| Spinel | Octahedra, twins | Often twinned (spinel law) |
| Garnet (all) | Dodecahedra, trapezohedra | Rhombic dodecahedron common |
| Fluorite | Cubes, octahedra | Perfect octahedral cleavage |
| Sodalite | Dodecahedra | Rare crystals |
| Pyrite | Cubes, pyritohedra | "Fool's gold" |

**Gemmological Significance**:
- Stays dark in polariscope (true isotropic)
- No dichroism in dichroscope
- Single RI reading on refractometer
- ADR (anomalous double refraction) may occur in strained crystals

## Tetragonal System

**Characteristics**:
- Two equal horizontal axes, one different vertical
- All axes at 90°
- Uniaxial (positive or negative)
- Dichroism possible

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Zircon | Prismatic, bipyramidal | Uniaxial + |
| Scapolite | Prismatic | Uniaxial - |
| Scheelite | Bipyramidal | Uniaxial + |
| Rutile | Prismatic, acicular | Uniaxial + |

**Gemmological Significance**:
- Two RI readings (ordinary and extraordinary)
- Uniaxial interference figure (Maltese cross)
- Zircon's high birefringence (0.059) causes visible doubling

## Hexagonal System

**Characteristics**:
- Three equal horizontal axes at 60°, one different vertical at 90°
- Uniaxial (positive or negative)
- Six-fold symmetry

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Beryl | Hexagonal prisms | Uniaxial - |
| Apatite | Hexagonal prisms | Uniaxial - |
| Taaffeite | Tabular | Uniaxial - |

**Gemmological Significance**:
- Hexagonal colour zoning (beryl, apatite)
- Characteristic crystal shapes
- Uniaxial interference figure

## Trigonal System

**Characteristics**:
- Similar to hexagonal but three-fold symmetry
- Often grouped with hexagonal (hexagonal division)
- Uniaxial

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Corundum | Barrel-shaped, tabular, bipyramidal | Uniaxial - |
| Tourmaline | Trigonal prisms, striated | Uniaxial - |
| Quartz | Hexagonal prisms with pyramidal terminations | Uniaxial + |
| Calcite | Rhombohedra | Uniaxial - |
| Rhodochrosite | Rhombohedra | Uniaxial - |

**Gemmological Significance**:
- Corundum shows hexagonal colour zoning
- Tourmaline's triangular cross-section
- Quartz prisms have horizontal striations

## Orthorhombic System

**Characteristics**:
- Three unequal axes at 90°
- Biaxial (positive or negative)
- Three principal RI values (α, β, γ)

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Topaz | Prismatic | Biaxial + |
| Peridot | Tabular | Biaxial + |
| Tanzanite | Prismatic | Biaxial + |
| Chrysoberyl | Tabular, cyclic twins | Biaxial + |
| Andalusite | Prismatic | Biaxial - |
| Iolite | Prismatic | Biaxial - |
| Sinhalite | Prismatic | Biaxial - |
| Kornerupine | Prismatic | Biaxial - |
| Danburite | Prismatic | Biaxial - |
| Prehnite | Tabular | Biaxial + |
| Enstatite | Prismatic | Biaxial + |
| Sillimanite | Prismatic, fibrous | Biaxial + |

**Gemmological Significance**:
- Three pleochroic colours possible (trichroism)
- Tanzanite shows very strong trichroism
- Iolite known as "Viking compass stone" due to strong pleochroism
- Biaxial interference figure (separate brushes)

## Monoclinic System

**Characteristics**:
- Three unequal axes
- Two axes at 90°, one oblique
- Biaxial
- Lower symmetry than orthorhombic

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Spodumene | Prismatic, flattened | Biaxial + |
| Orthoclase | Prismatic, tabular | Biaxial - |
| Jadeite | Massive (rarely crystals) | Biaxial + |
| Nephrite | Fibrous massive | Biaxial - |
| Malachite | Massive, botryoidal | Biaxial - |
| Diopside | Prismatic | Biaxial + |
| Epidote | Prismatic | Biaxial - |
| Sphene (Titanite) | Wedge-shaped | Biaxial + |
| Euclase | Prismatic | Biaxial + |
| Brazilianite | Prismatic | Biaxial + |

**Gemmological Significance**:
- Spodumene (kunzite) has perfect cleavage at 87°
- Jadeite usually seen as aggregate, not single crystals
- Sphene's high dispersion (0.051) exceeds diamond

## Triclinic System

**Characteristics**:
- Three unequal axes
- No 90° angles
- Lowest symmetry
- Biaxial

**Gem Examples**:
| Gemstone | Crystal Habit | Optic Sign |
|----------|---------------|------------|
| Labradorite | Tabular | Biaxial + |
| Amazonite | Prismatic | Biaxial - |
| Turquoise | Massive | Biaxial + |
| Rhodonite | Tabular, massive | Biaxial + |
| Axinite | Wedge-shaped | Biaxial - |
| Kyanite | Bladed | Biaxial - |

**Gemmological Significance**:
- Labradorite's lamellar twinning causes labradorescence
- Kyanite shows strong directional hardness variation
- Often form massive/aggregate rather than distinct crystals

## Amorphous Materials

**No crystal structure**:
| Material | Type | Notes |
|----------|------|-------|
| Opal | Mineraloid | Ordered silica spheres, but not crystalline |
| Glass | Artificial | No crystal structure |
| Obsidian | Natural glass | Volcanic origin |
| Moldavite | Tektite | Impact glass |
| Amber | Organic | Fossilised resin |

**Gemmological Significance**:
- Isotropic (singly refractive)
- No pleochroism
- May show strain patterns in polariscope

## Quick Identification by Crystal System

| Test | Cubic | Uniaxial | Biaxial | Amorphous |
|------|-------|----------|---------|-----------|
| Polariscope | Dark (ADR possible) | 4 blinks | 4 blinks | Dark or strain |
| Dichroscope | 1 colour | 2 colours | 2-3 colours | 1 colour |
| RI readings | 1 | 2 | 2-3 | 1 |
| Birefringence | 0 | Present | Present | 0 |
