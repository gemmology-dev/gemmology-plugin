# Gemstone Crystallographic Data

Reference data for crystal structures of common gemstones.

## Crystal Systems Overview

| System | Axes | Angles | Examples |
|--------|------|--------|----------|
| Cubic | a = b = c | α = β = γ = 90° | Diamond, spinel, garnet |
| Tetragonal | a = b ≠ c | α = β = γ = 90° | Zircon |
| Hexagonal | a = b ≠ c | α = β = 90°, γ = 120° | Beryl, apatite |
| Trigonal | a = b = c | α = β = γ ≠ 90° | Corundum, quartz, tourmaline |
| Orthorhombic | a ≠ b ≠ c | α = β = γ = 90° | Topaz, peridot, chrysoberyl |
| Monoclinic | a ≠ b ≠ c | α = γ = 90°, β ≠ 90° | Spodumene, orthoclase |
| Triclinic | a ≠ b ≠ c | α ≠ β ≠ γ ≠ 90° | Plagioclase, kyanite |

## Cleavage Related to Structure

| Gemstone | Cleavage | Structural Basis |
|----------|----------|------------------|
| Diamond | Perfect {111} | Weaker bonds between layers |
| Topaz | Perfect {001} | Hydroxyl/fluorine planes |
| Spodumene | Perfect {110} | Pyroxene chain structure |
| Feldspar | Perfect {001}, {010} | Sheet structure |
| Fluorite | Perfect {111} | Fluorine arrangement |
