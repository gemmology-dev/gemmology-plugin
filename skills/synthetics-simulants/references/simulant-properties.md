# Simulant Properties Guide

Identification data for gemstone simulants.

## Diamond Simulants

### Cubic Zirconia (CZ)

| Property | Value | vs Diamond |
|----------|-------|------------|
| Composition | ZrO₂ | C |
| RI | 2.15-2.18 | 2.42 |
| SG | 5.6-6.0 | 3.52 |
| Hardness | 8-8.5 | 10 |
| Dispersion | 0.060 | 0.044 |
| Thermal conductivity | Low | Very high |
| Optic character | Isotropic | Isotropic |

**Detection**:
- Heavier (SG test)
- Lower thermal conductivity (diamond tester fails)
- Higher dispersion (rainbow fire)
- May show orange flash
- Warm to touch (vs diamond cold)

### Moissanite

| Property | Value | vs Diamond |
|----------|-------|------------|
| Composition | SiC | C |
| RI | 2.648-2.691 | 2.42 |
| SG | 3.22 | 3.52 |
| Hardness | 9.25 | 10 |
| Dispersion | 0.104 | 0.044 |
| Birefringence | 0.043 | None |
| Thermal conductivity | High | Very high |

**Detection**:
- Doubling visible (birefringence)
- Extreme fire (high dispersion)
- May pass thermal tester
- Specific gravity lower
- Needle-like inclusions possible

### Synthetic Rutile

| Property | Value | vs Diamond |
|----------|-------|------------|
| Composition | TiO₂ | C |
| RI | 2.616-2.903 | 2.42 |
| SG | 4.26 | 3.52 |
| Hardness | 6-6.5 | 10 |
| Dispersion | 0.280 | 0.044 |
| Birefringence | 0.287 | None |

**Detection**:
- Extreme dispersion (rainbow)
- Very strong doubling
- Much softer
- Yellow tint common
- Historical simulant (1940s-50s)

## Quick Detection Reference

### By Test

| Test | What It Reveals |
|------|-----------------|
| RI | Distinguishes most simulants |
| SG | Confirms identity |
| Polariscope | Optic character |
| Microscope | Inclusions, structure |
| UV fluorescence | Characteristic reactions |
| Spectrum | Chromophore identification |
| Thermal probe | Diamond testing |

### Common Confusion Pairs

| Genuine | Common Simulant | Key Difference |
|---------|-----------------|----------------|
| Diamond | CZ | Thermal conductivity |
| Diamond | Moissanite | Birefringence |
| Ruby | Garnet | Optic character |
| Ruby | Red spinel | RI, optic character |
| Emerald | Tsavorite | RI (much higher) |
| Sapphire | Blue spinel | RI, fluorescence |
| Alexandrite | CC sapphire | RI, SG |
