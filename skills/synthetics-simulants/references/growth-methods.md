# Synthetic Growth Methods

Technical overview of laboratory gem growth processes.

## Flame-Fusion (Verneuil Process)

### Process Description

1. Powdered raw materials dropped through flame
2. Material melts and falls onto seed
3. Boule builds up gradually
4. Rotation creates cylindrical shape

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | ~2000°C (corundum) |
| Growth rate | Fast (hours) |
| Crystal shape | Cylindrical boule |
| Cost | Very low |
| Size | Up to 100+ carats |

### Materials Produced

| Synthetic | Dopants | Notes |
|-----------|---------|-------|
| Ruby | Cr₂O₃ | Most common |
| Blue sapphire | Fe₂O₃ + TiO₂ | Various shades |
| Star ruby/sapphire | TiO₂ (excess) | Creates silk |
| Spinel | Various | Different colours |
| Rutile | None | TiO₂ crystal |
| Strontium titanate | None | Diamond simulant |

### Identification Features

| Feature | Cause |
|---------|-------|
| Curved striae | Non-planar growth front |
| Gas bubbles | Trapped gases |
| Boule marks | Growth striations |
| Internal stress | Rapid cooling |

### Advantages and Limitations

| Advantages | Limitations |
|------------|-------------|
| Very low cost | Obvious inclusions |
| Large sizes | Curved striae |
| Fast production | Limited to certain materials |
| Well-established | Bubbles common |

## Flux Method

### Process Description

1. Raw materials dissolved in molten flux (solvent)
2. Slow cooling precipitates crystals
3. Crystals grow over weeks to months
4. Flux dissolved to recover crystals

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | 800-1300°C |
| Growth rate | Very slow (months) |
| Crystal shape | Natural-looking |
| Cost | Moderate to high |
| Size | Variable (often <10ct) |

### Common Fluxes

| Flux | Used For |
|------|----------|
| PbO-PbF₂ | Ruby, sapphire |
| Li₂O-MoO₃ | Emerald |
| B₂O₃ | Various |
| Na₂O | Various |

### Materials Produced

| Synthetic | Producer Examples |
|-----------|-------------------|
| Ruby | Kashan, Chatham, Knischka |
| Sapphire | Chatham |
| Emerald | Chatham, Gilson, Lennix |
| Alexandrite | Creative Crystals |
| Spinel | Rare flux production |

### Identification Features

| Feature | Cause |
|---------|-------|
| Flux veils | Trapped flux |
| Platinum crystals | From crucible |
| "Wispy" patterns | Solidified flux |
| Natural-looking growth | Slow crystallisation |
| Colour zoning | Temperature gradients |

### Advantages and Limitations

| Advantages | Limitations |
|------------|-------------|
| High quality | Slow/expensive |
| Natural-like inclusions | Size limited |
| Good colour | Flux residues |
| Fewer bubbles | Platinum inclusions |

## Hydrothermal Method

### Process Description

1. Nutrient material dissolves in hot water
2. Under high pressure in autoclave
3. Crystals grow on seed plates
4. Weeks to months of growth

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | 300-600°C |
| Pressure | 1000-3000 atm |
| Growth rate | Slow (weeks-months) |
| Cost | Moderate |
| Size | Variable |

### Materials Produced

| Synthetic | Notes |
|-----------|-------|
| Quartz | Major industrial use |
| Emerald | Biron, Regency, Russian |
| Ruby | Limited production |
| Sapphire | Limited production |
| Alexandrite | High quality |

### Identification Features

| Feature | Cause |
|---------|-------|
| Chevron growth | Zigzag growth pattern |
| Seed plate | Original seed visible |
| Nail-head spicules | Characteristic inclusions |
| Two-phase inclusions | May resemble natural |
| Angular zoning | Growth pattern |

### Advantages and Limitations

| Advantages | Limitations |
|------------|-------------|
| Close to natural growth | High pressure needed |
| Fewer obvious tells | Expensive equipment |
| Good clarity | Seed plate may show |
| Large sizes possible | Time-intensive |

## Czochralski (Pulled) Method

### Process Description

1. Raw material melted in crucible
2. Seed crystal dipped in melt
3. Slowly pulled upward with rotation
4. Crystal grows from melt surface

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | Above melting point |
| Growth rate | Moderate |
| Crystal shape | Cylindrical |
| Cost | Moderate |
| Size | Very large possible |

### Materials Produced

| Synthetic | Applications |
|-----------|--------------|
| Alexandrite | Gems, lasers |
| Ruby | Lasers, gems |
| Sapphire | Windows, gems |
| YAG | Lasers, former simulant |
| GGG | Former diamond simulant |

### Identification Features

| Feature | Cause |
|---------|-------|
| Very clean | Controlled growth |
| Subtle swirls | Rotation |
| Perfect colour | Precise chemistry |
| No flux/bubbles | Melt growth |

### Advantages and Limitations

| Advantages | Limitations |
|------------|-------------|
| Very clean crystals | Expensive equipment |
| Large sizes | Limited materials |
| Controlled composition | May be too perfect |
| Laser-quality | Subtle swirl marks |

## Skull Melt Method

### Process Description

1. Material in skull-shaped container
2. RF induction heating
3. Outer layer forms solid "skull"
4. Inner material melts and crystallises

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | Very high (2700°C for CZ) |
| Growth rate | Fast |
| Crystal shape | Irregular |
| Cost | Low for CZ |
| Size | Large |

### Materials Produced

| Synthetic | Notes |
|-----------|-------|
| Cubic zirconia | Primary gem product |
| Fabulite (YAG) | Historical |

### Identification Features

| Feature | Cause |
|---------|-------|
| Clean | Controlled environment |
| May have inclusions | Process variables |
| High dispersion | Material property (CZ) |

## CVD (Chemical Vapor Deposition)

### Process Description

1. Gas mixture (carbon source + hydrogen)
2. Plasma activation
3. Carbon deposits layer by layer
4. Diamond grows on substrate

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | 700-1200°C |
| Pressure | Low (vacuum) |
| Growth rate | Slow layer growth |
| Cost | Moderate-high |
| Size | Growing larger |

### Materials Produced

| Synthetic | Notes |
|-----------|-------|
| Diamond | Primary gem application |
| Silicon carbide | Industrial |

### Identification Features (Diamond)

| Feature | Notes |
|---------|-------|
| Layer growth | Visible with DiamondView |
| Strain patterns | Different from natural |
| Spectroscopy | Nitrogen-free typically |
| Inscriptions | Often laser inscribed |

## HPHT (High Pressure High Temperature)

### Process Description

1. Carbon source with metal catalyst
2. Extreme pressure and temperature
3. Mimics natural diamond formation
4. Crystals grow over days/weeks

### Technical Details

| Aspect | Specification |
|--------|---------------|
| Temperature | 1300-1600°C |
| Pressure | 50,000-70,000 atm |
| Growth rate | Days to weeks |
| Cost | High |
| Size | Growing larger |

### Materials Produced

| Synthetic | Notes |
|-----------|-------|
| Diamond | Various colours |
| Other high-pressure phases | Research |

### Identification Features (Diamond)

| Feature | Notes |
|---------|-------|
| Flux inclusions | Metal catalyst remnants |
| Unusual growth | Different from natural |
| Spectroscopy | Type IIa or IIb common |
| Magnetism | May attract magnet |

## Comparison Summary

| Method | Speed | Cost | Quality | Detection |
|--------|-------|------|---------|-----------|
| Flame-fusion | Fast | Low | Variable | Easy |
| Flux | Slow | High | High | Moderate |
| Hydrothermal | Slow | Moderate | High | Moderate |

| Skull melt | Fast | Low | Good | Easy |
| CVD | Slow | High | Very high | Lab needed |
| HPHT | Moderate | Very high | Very high | Lab needed |
