# Absorption Spectra Reference

Diagnostic absorption spectra for coloured gemstones, with wavelength positions in nanometres (nm).

## Reading the Spectrum

- **Red end**: 700nm
- **Violet end**: 400nm
- **Lines**: Sharp, narrow absorption bands
- **Bands**: Broad absorption regions
- **Cutoff**: Complete absorption of a spectral region

## Corundum Spectra

### Ruby (Chromium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Doublet** | 694.2, 692.8 | Fluorescent lines (emission, not absorption) |
| Weak line | 668 | Absorption |
| Weak line | 659 | Absorption |
| Broad band | 550 (centre) | Absorption in yellow-green |
| Lines | 476, 475, 468 | Weak blue absorption |

**Key diagnostic**: The 694/693nm doublet (appears bright in fluorescent rubies, dark line in iron-rich stones)

### Blue Sapphire (Iron/Titanium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Diagnostic trio** | 471, 460, 450 | Three bands in blue region |
| Band | 450 | Strong in Australian/Thai sapphires |

**Key diagnostic**: 450nm band strength indicates iron content. Kashmir sapphires show weak spectrum.

### Yellow Sapphire

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Band | 450 | Iron absorption |
| May show | 471, 460 | Weaker than blue sapphire |

### Padparadscha

Shows weak chromium features plus iron spectrum.

## Beryl Spectra

### Emerald (Chromium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Doublet** | 683, 680 | Chromium doublet (diagnostic) |
| Line | 662 | Weaker |
| Line | 646 | Weaker |
| Broad band | 600-630 | Partial absorption |
| Cutoff | Below 470 | Absorption of violet |

**Key diagnostic**: 683/680nm doublet confirms chromium colouring

### Aquamarine (Iron spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Band | 537 | Diagnostic if present |
| Band | 456 | May be present |

Spectrum is often weak or not visible.

## Garnet Spectra

### Almandine (Iron spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Strong bands** | 576, 527, 505 | Very diagnostic trio |
| Band | 462 | Weaker |
| Band | 438 | Weaker |
| Band | 428 | Weaker |
| Band | 404 | Weaker |

**Key diagnostic**: The 505, 527, 576nm bands are unmistakable

### Pyrope

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Broad band | 575 | Iron absorption |
| Lines | May show chromium | If chrome-pyrope |

### Demantoid (Chromium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Cutoff** | Below 443 | Diagnostic iron cutoff |
| Line | 701 | Chromium |
| Band | 640 | Chromium |
| Band | 622 | Chromium |
| Band | 443 | Iron |

**Key diagnostic**: Strong violet cutoff at 443nm

### Tsavorite (Vanadium/Chromium)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Broad bands | Red region | Chromium/vanadium absorption |
| Line | 430 | Variable |

### Spessartine (Manganese spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Bands | 495, 485, 462 | Manganese absorption |
| Band | 432 | Stronger |
| Band | 424 | Manganese |
| Band | 412 | Manganese |

**Key diagnostic**: Multiple bands in blue-violet region

### Rhodolite

Shows combination of almandine and pyrope spectra, typically weaker than pure almandine.

## Spinel Spectra

### Red Spinel (Chromium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **"Organ pipe"** | 686, 685, 684, 683, 682, 681 | Multiple chromium lines |
| Broad band | 540 (centre) | Absorption |
| Group | 656-632 | Variable lines |

**Key diagnostic**: The "organ pipe" group of lines in red is unique to spinel

### Blue Spinel (Cobalt or iron)

**Cobalt blue**:
| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Triple band** | 635, 580, 540 | Cobalt absorption |

**Iron blue**:
| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Band | 458 | Iron |

**Gahnite component**:
| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| Lines | 478, 443 | Zinc spinel |

## Zircon Spectra

### Typical Zircon (Uranium spectrum)

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Diagnostic line** | 653.5 | Very strong, diagnostic |
| Many lines | Throughout spectrum | Complex uranium spectrum |
| Lines | 691, 689, 659 | Variable strength |
| Lines | 562, 537, 516 | Variable strength |

**Key diagnostic**: The 653.5nm line is unmistakable

### Low (Metamict) Zircon

Spectrum becomes fuzzy or absent due to radiation damage.

## Peridot Spectrum

| Feature | Position (nm) | Description |
|---------|---------------|-------------|
| **Diagnostic trio** | 497, 495, 493 | Iron absorption |
| Bands | 473, 453 | Weaker iron bands |

**Key diagnostic**: Three evenly-spaced bands around 495nm

### Topaz

Generally no diagnostic spectrum (iron sometimes at 450nm region).

### Synthetic Indicators

**Flame-fusion synthetics**: Generally show cleaner spectra than natural
**Flux-grown**: May show dopant-specific lines
**Hydrothermal**: Often cleaner than natural
