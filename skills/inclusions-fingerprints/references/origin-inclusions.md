# Origin-Diagnostic Inclusions

Inclusion characteristics that help determine geographic origin.

| Inclusion | Description | Diagnostic Value |
|-----------|-------------|------------------|

**Key signature**: Marble-hosted genesis; calcite + fine silk + strong fluorescence

## Interpretation Guidelines

1. **No single inclusion is conclusive** - use pattern of features
2. **Treatment affects inclusions** - heating dissolves/alters some features
3. **Overlapping characteristics exist** - some origins similar
4. **Require multiple observations** - systematic examination essential
5. **Laboratory confirmation advised** - for valuable/important stones
6. **New deposits emerge regularly** - reference data needs updating
7. **Combine with other tests** - chemistry, fluorescence, spectra
