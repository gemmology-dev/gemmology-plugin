# Gemstone Species and Varieties

Comprehensive reference for major coloured gemstone species and their varieties. Organised by mineral group with key identification properties.

## Corundum (Al₂O₃)

**Crystal system**: Trigonal | **Hardness**: 9 | **SG**: 3.97-4.05 | **RI**: 1.762-1.770

### Ruby

| Property | Value |
|----------|-------|
| Colour cause | Chromium (Cr³⁺) |
| Ideal colour | "Pigeon blood" red |
| Key origins | Burma, Mozambique, Thailand, Madagascar |
| Fluorescence | Strong red (Burmese) |
| Common treatments | Heat, glass filling |

### Sapphire Varieties

| Variety | Colour | Notes |
|---------|--------|-------|
| Blue sapphire | Blue | Fe/Ti intervalence |
| Pink sapphire | Pink | Chromium + iron |
| Yellow sapphire | Yellow | Fe³⁺ or colour centres |
| Padparadscha | Pink-orange | Sri Lankan origin traditionally |
| White/colourless | Colourless | Rare naturally |
| Colour-change | Variable | Cr/V content |
| Star sapphire | Any colour | Asterism from silk |

**Padparadscha requirements**:
- Pinkish-orange to orangey-pink
- Medium saturation
- No predominantly pink or orange
- Originally Sri Lanka only (now debated)

## Beryl (Be₃Al₂Si₆O₁₈)

**Crystal system**: Hexagonal | **Hardness**: 7.5-8 | **SG**: 2.66-2.92 | **RI**: 1.565-1.602

### Varieties

| Variety | Colour | Colour Cause |
|---------|--------|--------------|
| Emerald | Green | Chromium, vanadium |
| Aquamarine | Blue | Iron (Fe²⁺) |
| Morganite | Pink | Manganese |
| Heliodor | Yellow | Iron (Fe³⁺) |
| Goshenite | Colourless | None |
| Red beryl | Red | Manganese (Mn³⁺) |
| Maxixe | Blue | Irradiation (unstable) |

### Emerald

| Property | Value |
|----------|-------|
| Clarity type | Type III (always included) |
| Key origins | Colombia, Zambia, Brazil, Ethiopia |
| Common treatments | Oiling/resin filling |
| "Jardin" | Garden of inclusions (characteristic) |

### Aquamarine

| Property | Value |
|----------|-------|
| Ideal colour | Pure blue, no green |
| Clarity type | Type I (usually eye-clean) |
| Treatment | Heat (removes green) |
| Origins | Brazil, Madagascar, Pakistan, Nigeria |

## Chrysoberyl (BeAl₂O₄)

**Crystal system**: Orthorhombic | **Hardness**: 8.5 | **SG**: 3.70-3.78 | **RI**: 1.744-1.755

### Varieties

| Variety | Phenomenon | Notes |
|---------|------------|-------|
| Chrysoberyl | None | Yellow-green typical |
| Alexandrite | Colour change | Green in daylight, red in incandescent |
| Cat's eye | Chatoyancy | "Cymophane"; most valuable cat's eye |

### Alexandrite

| Property | Value |
|----------|-------|
| Colour change | Green ↔ Red (ideal) |
| Cause | Chromium in specific crystal field |
| Best origin | Russia (Urals) - largely exhausted |
| Other origins | Brazil, Sri Lanka, Tanzania |
| Value factors | Strength and quality of colour change |

## Spinel (MgAl₂O₄)

**Crystal system**: Cubic | **Hardness**: 8 | **SG**: 3.58-3.61 | **RI**: 1.712-1.736

| Property | Notes |
|----------|-------|
| Optical character | Isotropic (singly refractive) |
| Pleochroism | None |
| Key advantage | No heat treatment typically |
| Colours | Red, pink, blue, purple, orange |
| Origins | Burma, Tanzania (Mahenge), Sri Lanka, Vietnam, Tajikistan |

**Historical note**: Many famous "rubies" are actually spinels (e.g., Black Prince's Ruby, Timur Ruby).

### Colour Varieties

| Colour | Notes |
|--------|-------|
| Red | Most valuable; Burmese finest |
| Pink | "Hot pink" Mahenge very desirable |
| Blue | Cobalt blue most valuable |
| Lavender | Attractive, more affordable |
| Flame (Jedi) | Vivid pink-red, highly valued |

## Garnet Group (X₃Y₂(SiO₄)₃)

**Crystal system**: Cubic | **Hardness**: 6.5-7.5 | **Optical**: Isotropic

### Species Overview

| Species | Formula | Colours | SG | RI |
|---------|---------|---------|-----|-----|
| Pyrope | Mg₃Al₂Si₃O₁₂ | Red | 3.65-3.87 | 1.730-1.760 |
| Almandine | Fe₃Al₂Si₃O₁₂ | Red-brown | 3.90-4.20 | 1.790-1.820 |
| Spessartine | Mn₃Al₂Si₃O₁₂ | Orange | 4.12-4.20 | 1.790-1.810 |
| Grossular | Ca₃Al₂Si₃O₁₂ | Various | 3.57-3.73 | 1.730-1.760 |
| Andradite | Ca₃Fe₂Si₃O₁₂ | Green, yellow | 3.82-3.85 | 1.880-1.889 |
| Uvarovite | Ca₃Cr₂Si₃O₁₂ | Green | 3.77 | 1.860 |

### Important Varieties

| Variety | Parent Species | Colour | Notes |
|---------|----------------|--------|-------|
| **Rhodolite** | Pyrope-Almandine | Purple-red | Most popular garnet |
| **Tsavorite** | Grossular | Green | Rivals emerald; Kenya, Tanzania |
| **Demantoid** | Andradite | Green | Highest dispersion; horsetail inclusions |
| **Hessonite** | Grossular | Orange | "Roiled" appearance |
| **Mandarin** | Spessartine | Vivid orange | Very saturated |
| **Malaya** | Pyrope-Spessartine | Pink-orange | Originally rejected material |
| **Mali** | Grossular-Andradite | Yellow-green | High dispersion |
| **Colour-change** | Various | Variable | Rare, valuable |

## Tourmaline Group

**Crystal system**: Trigonal | **Hardness**: 7-7.5 | **RI**: 1.616-1.650

### Species

| Species | Composition | Colours |
|---------|-------------|---------|
| Elbaite | Li, Al | Most gem colours |
| Dravite | Mg | Brown, yellow |
| Schorl | Fe | Black |
| Liddicoatite | Ca, Li | Multicoloured |

### Colour Varieties

| Name | Colour | Notes |
|------|--------|-------|
| Rubellite | Pink-red | Chromium-coloured |
| Indicolite | Blue | Iron content |
| Chrome tourmaline | Green | Chromium (not vanadium) |
| Paraiba | Blue-green | Copper-bearing; electric colour |
| Watermelon | Pink centre, green rim | Highly collectible |
| Canary | Yellow | Iron-coloured |

### Paraiba Tourmaline

| Property | Value |
|----------|-------|
| Colour cause | Copper + manganese |
| Colour | "Neon" blue to green |
| Original source | Paraiba, Brazil (1989) |
| Other sources | Nigeria, Mozambique |
| Value | Among highest for tourmaline |

## Other Important Species

### Topaz (Al₂SiO₄(F,OH)₂)

**Hardness**: 8 | **SG**: 3.49-3.57 | **RI**: 1.609-1.643

| Variety | Colour | Notes |
|---------|--------|-------|
| Imperial topaz | Orange, pink-orange | Most valuable; Ouro Preto, Brazil |
| Blue topaz | Blue | Irradiated/heated (all) |
| Pink topaz | Pink | Natural or heated imperial |
| Sherry topaz | Brownish-orange | Natural |
| White topaz | Colourless | Common |

### Zircon (ZrSiO₄)

**Hardness**: 6-7.5 | **SG**: 3.9-4.8 | **RI**: 1.78-1.99

| Type | RI | Birefringence | Notes |
|------|-----|---------------|-------|
| High | 1.925-1.984 | 0.059 | Strong doubling visible |
| Medium | 1.84-1.90 | 0.040 | Intermediate |
| Low (metamict) | 1.78-1.85 | ~0 | Radiation damage |

**Not to be confused with cubic zirconia (synthetic)**

### Peridot ((Mg,Fe)₂SiO₄)

**Hardness**: 6.5-7 | **SG**: 3.27-3.48 | **RI**: 1.650-1.703

| Property | Notes |
|----------|-------|
| Colour | Only green (idiochromatic) |
| Birefringence | High (0.036) - visible doubling |
| Diagnostic | "Lily pad" inclusions |
| Origins | Burma, Pakistan, Arizona, China |

### Tanzanite (Ca₂Al₃(SiO₄)(Si₂O₇)O(OH))

**Hardness**: 6-7 | **SG**: 3.35 | **RI**: 1.691-1.700

| Property | Notes |
|----------|-------|
| Single source | Merelani Hills, Tanzania |
| Colour | Violetish-blue (heated) |
| Natural colour | Brown (typically heated) |
| Trichroism | Very strong (blue, purple, burgundy) |
| Cleavage | Perfect in one direction |

### Opal (SiO₂·nH₂O)

**Hardness**: 5.5-6.5 | **SG**: 1.98-2.25 | **RI**: 1.37-1.52

| Type | Body Colour | Play-of-Colour |
|------|-------------|----------------|
| Black opal | Dark | Yes (most valuable) |
| White opal | Light | Yes |
| Crystal opal | Transparent | Yes |
| Fire opal | Orange | May or may not have |
| Boulder opal | Ironstone matrix | Yes |
