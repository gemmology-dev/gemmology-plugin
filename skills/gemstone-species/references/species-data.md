### Sphene (Titanite)

| Property | Value |
|----------|-------|
| Formula | CaTiSiO₅ |
| Crystal System | Monoclinic |
| RI | 1.900-2.034 |
| Birefringence | 0.100-0.135 |
| SG | 3.52 (3.48-3.60) |
| Hardness | 5-5.5 |
| Dispersion | 0.051 (extreme - higher than diamond) |
| Optic Sign | Biaxial (+) |

**Key features**:
- Highest dispersion of any gem (fire exceeds diamond)
- Strong birefringence causes pronounced doubling
- Relatively soft - collector stone only
- Yellow, green, brown colours
- Often included

### Kyanite

| Property | Value |
|----------|-------|
| Formula | Al₂SiO₅ |
| Crystal System | Triclinic |
| RI | 1.712-1.734 |
| Birefringence | 0.015-0.033 |
| SG | 3.68 (3.65-3.69) |
| Hardness | 4-4.5 (across crystal), 6-7 (along crystal) |
| Optic Sign | Biaxial (-) |

**Key features**:
- **Directional hardness** - unique property
- Blue colour, sometimes green
- Polymorph of andalusite and sillimanite
- Difficult to cut due to hardness variation
- Perfect cleavage
