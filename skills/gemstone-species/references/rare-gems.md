## Detailed Rare Species Profiles

### Taaffeite

| Property | Value |
|----------|-------|
| Formula | BeMgAl₄O₈ |
| Crystal System | Hexagonal |
| RI | 1.719-1.724 |
| Birefringence | 0.004-0.005 |
| SG | 3.61 |
| Hardness | 8-8.5 |
| Optic Sign | Uniaxial (-) |

**Identification**:
- Easily confused with spinel
- Birefringence distinguishes from spinel
- Check with polariscope carefully
- Rare faceted material

**Colours**: Mauve, pink, violet, red, colourless

### Painite

| Property | Value |
|----------|-------|
| Formula | CaZrBAl₉O₁₈ |
| Crystal System | Hexagonal |
| RI | 1.787-1.816 |
| Birefringence | 0.029 |
| SG | 4.01 |
| Hardness | 8 |
| Optic Sign | Uniaxial (-) |

**History**:
- Once listed as world's rarest mineral
- More crystals found since 2005
- Still exceptionally rare faceted
- Myanmar only significant source

**Colours**: Red-brown, orange-brown, red

### Benitoite

| Property | Value |
|----------|-------|
| Formula | BaTiSi₃O₉ |
| Crystal System | Hexagonal |
| RI | 1.757-1.804 |
| Birefringence | 0.047 |
| SG | 3.68 |
| Hardness | 6-6.5 |
| Optic Sign | Uniaxial (+) |
| Dispersion | 0.039-0.046 |

**Special features**:
- California state gem
- Strong blue fluorescence (SW UV)
- High dispersion
- Only gem-quality source: San Benito County, CA

**Colours**: Blue (violet-blue typical)

### Grandidierite

| Property | Value |
|----------|-------|
| Formula | (Mg,Fe)Al₃(BO₄)(SiO₄)O₂ |
| Crystal System | Orthorhombic |
| RI | 1.590-1.623 |
| Birefringence | 0.033 |
| SG | 2.98 |
| Hardness | 7.5 |
| Optic Sign | Biaxial (-) |

**Identification**:
- Strong pleochroism (blue-green, colourless, dark green)
- Resembles blue-green tourmaline
- Madagascar primary source
- Transparent material rare

**Colours**: Blue-green, green (best = blue-green)

### Jeremejevite

| Property | Value |
|----------|-------|
| Formula | Al₆B₅O₁₅(F,OH)₃ |
| Crystal System | Hexagonal |
| RI | 1.640-1.653 |
| Birefringence | 0.007-0.013 |
| SG | 3.28 |
| Hardness | 6.5-7.5 |
| Optic Sign | Uniaxial (-) |

**Sources**:
- Namibia (most gem-quality)
- Russia (original discovery)
- Germany, Tajikistan

**Colours**: Blue, colourless, yellow (blue most valued)

### Serendibite

| Property | Value |
|----------|-------|
| Formula | Ca₂(Mg,Al)₆(Si,Al,B)₆O₂₀ |
| Crystal System | Triclinic |
| RI | 1.701-1.706 |
| Birefringence | 0.005 |
| SG | 3.42 |
| Hardness | 6-7 |
| Optic Sign | Biaxial (-) |

**Notes**:
- Named after Serendib (Sri Lanka)
- Faceted gems extremely rare
- Most material is cabochon-grade
- Myanmar produced some facetable

**Colours**: Blue-green, black, grey-green

### Musgravite

| Property | Value |
|----------|-------|
| Formula | Be(Mg,Fe,Zn)₂Al₆O₁₂ |
| Crystal System | Trigonal |
| RI | 1.719-1.726 |
| Birefringence | 0.007 |
| SG | 3.68 |
| Hardness | 8-8.5 |
| Optic Sign | Uniaxial (-) |

**Relationship to taaffeite**:
- Same family (magnesiotaaffeite series)
- Mg-dominant = musgravite
- Very similar properties
- Both extremely rare

**Colours**: Grey-green, purple, violet

### Red Beryl (Bixbite)

| Property | Value |
|----------|-------|
| Formula | Be₃Al₂Si₆O₁₈ (with Mn³⁺) |
| Crystal System | Hexagonal |
| RI | 1.564-1.574 |
| Birefringence | 0.008 |
| SG | 2.66-2.70 |
| Hardness | 7.5-8 |
| Optic Sign | Uniaxial (-) |

**Unique factors**:
- Only gem source: Wah Wah Mountains, Utah
- Almost all material included
- Rarely exceeds 1 carat clean
- Rarer than diamond

**Colours**: Red to raspberry red

### Poudretteite

| Property | Value |
|----------|-------|
| Formula | KNa₂B₃Si₁₂O₃₀ |
| Crystal System | Hexagonal |
| RI | 1.511-1.532 |
| Birefringence | 0.016-0.021 |
| SG | 2.51 |
| Hardness | 5 |
| Optic Sign | Uniaxial (-) |

**Extreme rarity**:
- First found in Quebec, Canada
- Facetable gems from Myanmar
- Very few faceted stones exist
- Soft (5 Mohs) limits use

## Collecting Considerations

### Authentication

| Factor | Importance |
|--------|------------|
| Laboratory certificate | Essential for rare gems |
| Provenance documentation | Adds confidence |
| Species confirmation | Critical (many misidentifications) |
| Treatment status | Usually expected untreated |

### Value Factors

| Factor | Impact |
|--------|--------|
| Rarity | Primary driver |
| Size | Exponential for rare species |
| Colour | Species-appropriate ideal |
| Clarity | Less critical for very rare |
| Cut quality | Important for display |
| Origin | Premium for type localities |

### Common Misidentifications

| Rare Gem | Commonly Confused With |
|----------|----------------------|
| Taaffeite | Spinel |
| Alexandrite | Colour-change garnet, synthetic |
| Grandidierite | Tourmaline, apatite |
| Benitoite | Sapphire, iolite |
| Red beryl | Rubellite, red spinel |
| Paraíba | Blue topaz, apatite |

### Acquisition Tips

| Tip | Reason |
|-----|--------|
| Buy from specialists | Knowledge and authentication |
| Require certificates | Essential for value |
| Accept smaller sizes | Large rare gems extremely costly |
| Consider value trends | Some appreciate significantly |
| Verify origin claims | Premiums for type localities |
| Insurance appraisal | Protect investment |
