# Ruby Geographic Origins

Comprehensive guide to ruby sources and their characteristics.

## Major Origins

### Myanmar (Burma)

#### Mogok Valley

**Status**: Historic premier source, considered finest quality
**Geological setting**: Marble-hosted, contact metamorphic

| Characteristic | Description |
|----------------|-------------|
| **Colour** | "Pigeon blood" red, slightly bluish-red |
| **Saturation** | High, vivid |
| **Fluorescence** | Strong red (LW and SW) |
| **Silk** | Fine rutile needles, intact = unheated |
| **Key inclusions** | Calcite, apatite, sphene, spinel |
| **Iron content** | Very low |

**Market significance**:
- Commands highest premiums
- "Burma ruby" is marketing gold standard
- Unheated Mogok = exceptional value
- Certificates from major labs essential

#### Mong Hsu

**Status**: Major production since 1990s
**Geological setting**: Marble-hosted

| Characteristic | Description |
|----------------|-------------|
| **Colour (unheated)** | Dark cores, purplish-red rims |
| **Colour (heated)** | Pure red after treatment |
| **Treatment** | Almost all heat-treated |
| **Key inclusions** | Blue cores, boehmite needles |
| **Fluorescence** | Moderate to strong red |

**Market significance**:
- Large volumes available
- Lower value than Mogok
- Heat treatment expected and accepted
- Rarely sold as unheated

### Thailand/Cambodia

#### Chanthaburi-Trat (Thailand)

**Status**: Historical producer, now largely depleted
**Geological setting**: Basalt-hosted

| Characteristic | Description |
|----------------|-------------|
| **Colour** | Dark red, garnet-red, brownish tones |
| **Saturation** | Lower than Burmese |
| **Fluorescence** | Weak to inert (high iron) |
| **Inclusions** | Pyrrhotite, sparse silk |
| **Iron content** | High |

#### Pailin (Cambodia)

**Status**: Limited current production
**Geological setting**: Basalt-hosted

| Characteristic | Description |
|----------------|-------------|
| **Colour** | Dark red, similar to Thai |
| **Market** | Often grouped with Thai rubies |

**Market significance**:
- "Siam ruby" historically prestigious
- Now less valued than Burmese
- Dark colour considered less desirable
- Good for lower price points

### Mozambique

**Status**: Major current producer since 2009
**Geological setting**: Amphibolite-hosted

| Characteristic | Description |
|----------------|-------------|
| **Colour** | Ranges from pinkish to vivid red |
| **Quality range** | Wide, includes fine material |
| **Fluorescence** | Moderate red |
| **Key inclusions** | Amphibole, mica, negative crystals |
| **Iron content** | Moderate |

**Major deposits**:
- Montepuez (largest current producer)
- Ruambeze
- M'sawize

**Market significance**:
- Now supplies majority of world rubies
- Fine Mozambique rivals Burmese
- More affordable than Burma at equivalent quality
- Lab certification important for origin claims

### Madagascar

**Status**: Variable production from multiple localities
**Geological setting**: Various (marble, basalt, amphibolite)

| Locality | Characteristics |
|----------|-----------------|
| **Andilamena** | Marble-hosted, Mogok-like |
| **Vatomandry** | Good quality, variable |
| **Didy** | Pink to red, smaller sizes |
| **Ilakaka** | Alluvial, mixed quality |

**Market significance**:
- Quality varies greatly by deposit
- Fine material can rival Burma
- Often under-appreciated in market
- Andilamena most sought after

## Origin Determination Criteria

### Key Differentiators

| Origin | Iron Level | Fluorescence | Key Inclusions |
|--------|------------|--------------|----------------|
| Burma (Mogok) | Very low | Strong | Calcite, fine silk |
| Burma (Mong Hsu) | Low | Moderate-strong | Blue cores |
| Thai/Cambodian | High | Weak/inert | Pyrrhotite |
| Mozambique | Moderate | Moderate | Amphibole, mica |
| Vietnam | Low | Moderate-strong | Similar to Burma |
| Madagascar | Variable | Variable | Locality-specific |

### Testing Approach

1. **Visual assessment** - Colour, tone, saturation
2. **UV fluorescence** - LW and SW reaction
3. **Microscopy** - Inclusion examination
4. **Spectroscopy** - Absorption pattern
5. **Advanced analysis** - LA-ICP-MS for trace elements
