# Treatment Detection Guide

Methods for identifying common gemstone treatments.

## Heat Treatment

### Corundum (Ruby and Sapphire)

#### Signs of Low-Temperature Heating

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Dissolved silk | Partially dissolved rutile | Microscopy |
| Dotted silk | Broken needle patterns | Microscopy (40x+) |
| Stress halos | Around crystal inclusions | Darkfield |
| Discoid fractures | Disc-shaped cracks | Microscopy |

#### Signs of High-Temperature Heating

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Completely dissolved silk | No visible needles | Microscopy |
| Melted inclusions | Glassy, rounded | Microscopy |
| Sintered surfaces | Pitted, frosted | Surface examination |
| Chalky fluorescence | Patchy UV reaction | UV lamp |
| Altered inclusions | Changed appearance | Microscopy |
| Recrystallised boehmite | Short, stubby needles | Microscopy |
| Color zoning changes | Diffuse boundaries | Immersion |
| Healed fractures | With glass residues | Microscopy |

#### Detection Methods

| Method | What It Shows |
|--------|---------------|
| Microscopy | Inclusion alterations |
| UV fluorescence | Chalky zones, changes |
| FTIR spectroscopy | Hydroxyl changes (3309cm⁻¹) |
| Raman spectroscopy | Inclusion identification |

### Beryl (Emerald, Aquamarine)

#### Aquamarine Heat Treatment

| Before | After | Change |
|--------|-------|--------|
| Yellow-green | Blue | Fe³⁺ → Fe²⁺ reduction |
| Greenish-blue | Blue | Yellow component removed |

**Detection**: Essentially undetectable; industry-accepted

#### Emerald Oiling Detection

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Flash effect | Orange flash in fractures | Fibre-optic lighting |
| Sweating | Oil appearing at surface | Heat or solvent test |
| Fluorescence | Some fillers fluoresce | UV lamp |
| Bubbles in filler | Gas bubbles in resin | Microscopy |
| Different RI in fractures | Optical difference | Darkfield |

**Detection Methods**:
| Method | What It Detects |
|--------|-----------------|
| Microscopy | Flash effect, bubbles |
| UV fluorescence | Some resin fillers |
| FTIR spectroscopy | Filler identification (oil vs resin) |
| Raman spectroscopy | Filler composition |

### Zircon Heat Treatment

| Original | Heated Result | Detection |
|----------|---------------|-----------|
| Brown | Blue or colourless | Very difficult |
| Reddish | Colourless | Very difficult |

**Notes**: Universally treated; considered permanent; rarely detected

### Tanzanite Heat Treatment

| Before | After | Detection |
|--------|-------|-----------|
| Brown/burgundy | Blue-violet | Very difficult |

**Notes**: Almost all tanzanite is heated; unheated rare and premium-priced

## Diffusion Treatment

### Beryllium (Be) Diffusion (Sapphire)

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Surface colour concentration | Colour follows facet shape | Immersion in methylene iodide |
| Colour at girdle | Intense colour at thin areas | Magnification |
| No natural zoning | Colour follows cut, not crystal | Immersion |
| Orange rim on blue | Be-diffused blue | Immersion |

**Detection Methods**:
| Method | What It Shows |
|--------|---------------|
| Immersion microscopy | Colour distribution |
| LA-ICP-MS | Beryllium detection |
| SIMS | Beryllium profiling |

**Notes**:
- Light element Be requires advanced analysis
- Deep Be diffusion ("lattice diffusion") harder to detect
- May show natural-looking colour if deep

### Titanium Diffusion (Sapphire)

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Surface colour | Colour confined to surface | Immersion |
| Facet-related | Colour follows cut | Immersion |
| Blue rim | On colourless body | Immersion |

**Notes**: Shallow treatment; easier to detect than Be diffusion

### Cobalt Diffusion (Corundum)

| Indicator | Description |
|-----------|-------------|
| Intense blue colour | Vivid, even |
| Surface concentration | Colour at surface |
| Facet junctions | Colour accumulates |

## Fracture Filling

### Glass Filling (Ruby/Sapphire)

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Gas bubbles | In filled fractures | Microscopy |
| Flow structures | Glass flow patterns | Microscopy |
| Flash effect | Blue/orange flash | Reflected light |
| Different lustre | In filled areas | Surface examination |
| Collapsed areas | Where glass has degraded | Microscopy |
| Dendritic patterns | In filled fractures | Microscopy |

**Detection Methods**:
| Method | What It Shows |
|--------|---------------|
| Microscopy | Bubbles, flow, flash |
| EDXRF | Lead content (some glasses) |
| FTIR | Glass composition |
| Raman | Glass vs natural |

**Lead-glass filled rubies**:
- Very common in commercial material
- Significant value impact
- Can be unstable (acid-sensitive)
- Requires disclosure

### Resin/Polymer Filling (Emerald)

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Flash effect | Orange-blue flash | Fibre-optic |
| UV fluorescence | Yellow-green glow | UV lamp (some resins) |
| Bubbles | Gas bubbles in filler | Microscopy |
| Surface evidence | Filler at surface breaks | Microscopy |

**Filler Identification by FTIR**:
| Filler | Characteristic |
|--------|----------------|
| Oil (natural) | Organic peaks |
| Cedarwood oil | Specific pattern |
| Opticon | Polymer pattern |
| Permasafe | Resin pattern |
| Gematrat | Specific pattern |

## Coating

### Surface Coating Detection

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Scratches revealing base | Different colour beneath | Microscopy |
| Bubbles in coating | Gas bubbles | Microscopy |
| Uneven colour | Patchy distribution | Visual |
| Different lustre | Areas of different sheen | Visual |
| Edge wear | Coating worn at facet edges | Microscopy |

**Commonly Coated**:
- Topaz (blue coating on colourless)
- Tanzanite (coating to intensify)
- Diamond (for fancy colours)
- Quartz (various effects)

### Detection Tests

| Test | Method |
|------|--------|
| Microscope examination | Surface features |
| Scratch test (careful) | Reveals substrate |
| Solvent test | May dissolve some coatings |
| Hot point test | May bubble coating |

## Irradiation

### Colour Changes by Irradiation

| Gemstone | Before | After |
|----------|--------|-------|
| Topaz | Colourless | Blue (requires heat after) |
| Tourmaline | Pale pink | Intense pink |
| Diamond | Various | Fancy colours |
| Beryl (Maxixe) | Pale | Deep blue (unstable) |
| Pearl | White | Dark |
| Kunzite | Pale | More intense |

### Detection Challenges

| Stone | Detection Difficulty | Notes |
|-------|---------------------|-------|
| Blue topaz | Very difficult | Industry-standard treatment |
| Pink tourmaline | Difficult | May fade test help |
| Coloured diamond | Advanced testing | Spectroscopy, PL |
| Maxixe beryl | Easier | Fades in light |

**Detection Methods**:
| Method | Application |
|--------|-------------|
| Spectroscopy | Specific absorption patterns |
| Photoluminescence | Diamond colour centres |
| Fading test | Unstable colours (Maxixe) |
| Lab certification | Recommended for diamonds |

## Bleaching

### Pearl Bleaching

| Indicator | Description |
|-----------|-------------|
| Even colour | Unnaturally uniform |
| Weakened nacre | More fragile |
| Chalky surface | Damaged nacre |

**Detection**: Microscopy of nacre structure; experienced eye

### Jade Bleaching

| Indicator | Description | Detection |
|-----------|-------------|-----------|
| Surface texture | Etched appearance | Microscopy |
| Enhanced transparency | Acid removes impurities | Visual |
| Often followed by polymer filling | Creates "B-jade" | FTIR |

## Impregnation

### Stabilisation Treatment

| Material | Treatment | Purpose |
|----------|-----------|---------|
| Turquoise | Plastic/resin | Improve durability |
| Opal | Plastic | Stabilise |
| Lapis lazuli | Wax/plastic | Seal surface |
| Porous stones | Various | Enhance durability |

### Detection

| Method | What It Shows |
|--------|---------------|
| Hot point test | Plastic melts/smells |
| Specific gravity | Lower than untreated |
| UV fluorescence | Some plastics fluoresce |
| FTIR spectroscopy | Polymer identification |

## When to Suspect Treatment

| Observation | Consider |
|-------------|----------|
| Price too good | Heavy treatment |
| Unusually clean | Filling possible |
| Perfect colour | Enhancement |
| Unusual fluorescence | Various treatments |
| Surface anomalies | Coating, filling |
| Inconsistent features | Multiple treatments |
