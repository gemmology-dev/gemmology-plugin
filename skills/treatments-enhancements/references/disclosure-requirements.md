# Treatment Disclosure Requirements

Industry standards and legal requirements for gemstone treatment disclosure.

## Fundamental Principles

### General Rule

**All treatments must be disclosed at the point of sale**

Exceptions:
- Minor treatments considered normal for the species (minor oiling of emeralds)
- But even "standard" treatments should be disclosed upon request

### Levels of Disclosure

| Level | What's Disclosed | When Required |
|-------|------------------|---------------|
| Full | Treatment type, degree, permanence | Always recommended |
| Standard | Treatment status (treated/untreated) | Minimum legal requirement |
| Upon request | Additional details | When customer asks |

## Industry Standards

### CIBJO (World Jewellery Confederation)

**Blue Book requirements**:

| Category | Disclosure Requirement |
|----------|----------------------|
| A - Untreated | No disclosure needed if asked; should state "untreated" |
| B - Traditional, permanent | Disclose treatment type |
| C - Traditional, not permanent | Disclose treatment and maintenance needs |
| D - Not traditional, affects identification | Full disclosure essential |
| E - Surface modifications | Full disclosure; describe effect |

### American Gem Trade Association (AGTA)

**Treatment Codes**:

| Code | Meaning | Example |
|------|---------|---------|
| N | Not treated | Natural ruby |
| E | Enhanced | General enhancement |
| H | Heated | Heat-treated sapphire |
| U | Diffusion | Ti or Be diffusion |
| F | Filling | Fracture filling |
| R | Irradiation | Blue topaz |
| C | Coating | Coated topaz |
| B | Bleaching | Bleached jade |
| I | Impregnation | Stabilised turquoise |
| W | Waxing/oiling | Oiled emerald |
| D | Dyeing | Dyed pearl |

### GIA (Gemological Institute of America)

**Reporting system**:

| Report Statement | Meaning |
|------------------|---------|
| "No indication of heating" | Unheated |
| "Indications of heating" | Heat-treated |
| "Clarity enhanced" | Filling present |
| "Minor clarity enhancement" | Light treatment |
| "Moderate clarity enhancement" | Medium treatment |
| "Significant clarity enhancement" | Heavy treatment |

## Requirements by Gemstone

### Ruby

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Heating only | Required | Standard treatment |
| Flux healing | Required | Residues in fractures |
| Lead-glass filling | Critical | Major value impact |
| Beryllium diffusion | Critical | Affects value significantly |
| No treatment | Should state | Premium value |

**Special concerns**:
- Lead-glass filling has durability issues
- Disclosure essential for consumer protection

### Sapphire

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Heating only | Required | Very common |
| Ti diffusion | Required | Surface treatment |
| Be diffusion | Critical | Market confusion exists |
| No treatment | Should state | Premium value |

**Be-diffusion notes**:
- Sometimes difficult to detect
- Lab certification recommended

### Emerald

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Minor oil | Required | Almost universal |
| Moderate oil | Required | Common |
| Resin filling | Required | More permanent |
| Significant filling | Critical | Major value impact |
| No oil/treatment | Should state | Rare, very premium |

**Standard practice**:
- Minor oiling often expected
- Degree of treatment affects value

### Tanzanite

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Heating | Standard disclosure | Almost all are heated |
| No treatment | Should state | Very rare, premium |

### Blue Topaz

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Irradiation | Should disclose | Industry standard |
| Coating | Required | Less permanent |

**Industry practice**: Irradiation so common it may not be mentioned

### Turquoise

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Stabilised | Required | Very common |
| Reconstituted | Required | Ground and resin-bonded |
| Dyed | Required | Colour enhanced |
| Natural | Should state | Premium value |

### Pearl

| Treatment | Disclosure Level | Notes |
|-----------|------------------|-------|
| Bleaching | May disclose | Very common for white |
| Dyeing | Required | Colour change |
| Irradiation | Required | Creates dark colours |
| Coating | Required | Surface treatment |

## Legal Requirements

### United States (FTC Guidelines)

**16 CFR Part 23 - Guides for the Jewelry, Precious Metals, and Pewter Industries**:

| Requirement | Description |
|-------------|-------------|
| Material effect disclosure | Treatments affecting value must be disclosed |
| Deceptive practices prohibited | Cannot misrepresent treatment status |
| Laboratory-grown disclosure | Must disclose if not natural |

**Specific requirements**:
- Terms like "treated," "enhanced" required for significant treatments
- "Natural" implies no treatment
- Filling treatments must be disclosed

### United Kingdom

**Consumer Protection from Unfair Trading Regulations 2008**:

| Requirement | Description |
|-------------|-------------|
| Material information | Must not omit material facts |
| Misleading actions | Cannot mislead about characteristics |
| Average consumer standard | What a typical buyer would expect |

### European Union

**Various consumer protection directives require**:
- Full disclosure of treatment status
- Information about durability and care
- Accurate representation of value factors

## Verbal and Written Disclosure

### Point of Sale

| Method | Requirement |
|--------|-------------|
| Verbal explanation | Always |
| Written on receipt | Recommended |
| Certificate provided | Best practice |
| Care instructions | For relevant treatments |

### Invoice Language Examples

**Good practice**:
```
Ruby, 2.15ct, heat-treated
Emerald, 1.52ct, minor oil enhancement
Sapphire, 3.02ct, no indications of treatment (certified)
Blue topaz, 4.25ct, irradiated (standard industry treatment)
```

**Insufficient**:
```
Ruby, 2.15ct (no treatment information)
Gemstone, blue (vague, no treatment info)
```

## Consumer Rights

### What Buyers Should Expect

| Right | Description |
|-------|-------------|
| Full disclosure | All treatments revealed |
| Written documentation | Treatment on receipt |
| Return rights | If treatment not disclosed |
| Accurate pricing | Treatment reflected in price |

### Questions to Ask

| Question | Why Important |
|----------|---------------|
| "Is this stone treated?" | Basic information |
| "What treatment was done?" | Specific details |
| "Is the treatment permanent?" | Durability |
| "Does it require special care?" | Maintenance |
| "Can I get a certificate?" | Independent verification |

## Trade Consequences

### Non-Disclosure Risks

| Risk | Consequence |
|------|-------------|
| Legal action | Consumer protection violations |
| Return demands | Must accept returns |
| Reputation damage | Industry and consumer trust |
| Trade sanctions | Industry body actions |
| Financial loss | Refunds, legal costs |

### Best Practices for Sellers

| Practice | Benefit |
|----------|---------|
| Always disclose | Legal protection |
| Document everything | Paper trail |
| Use lab certificates | Independent verification |
| Train staff | Consistent disclosure |
| Written policies | Clear standards |

## Special Situations

### Estate and Antique Jewellery

| Situation | Approach |
|-----------|----------|
| Unknown treatment | State "treatment status unknown" |
| Period-appropriate treatment | Disclose if detectable |
| Unable to test | Recommend certification |

### Online Sales

| Requirement | Implementation |
|-------------|----------------|
| Clear description | Treatment in listing |
| Photos | Show any visible indicators |
| Return policy | Allow for non-disclosure claims |
| Certification | Provide or reference |

### Auctions

| Requirement | Standard |
|-------------|----------|
| Catalogue description | Treatment status |
| Condition report | Treatment details |
| Pre-sale viewing | Inspection opportunity |
| Certificate available | For significant items |
