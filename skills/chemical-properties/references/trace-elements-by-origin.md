# Trace Elements by Geographic Origin

Chemical fingerprints and trace element profiles for origin determination.

## Overview

Trace element analysis uses techniques like:
- **LA-ICP-MS** (Laser Ablation Inductively Coupled Plasma Mass Spectrometry)
- **EDXRF** (Energy Dispersive X-Ray Fluorescence)
- **LIBS** (Laser-Induced Breakdown Spectroscopy)

These detect elements in parts per million (ppm) or parts per billion (ppb).

## Analysis Methods Comparison

| Method | Elements Detected | Detection Limit | Sample Damage |
|--------|-------------------|-----------------|---------------|
| LA-ICP-MS | Most elements | ppb-ppm | Minimal (laser spot) |
| EDXRF | Heavy elements | ppm | None |
| LIBS | Most elements | ppm | Minimal |
| UV-Vis-NIR | Chromophores | % | None |

## Interpretation Guidelines

1. **Never rely on single element** - use ratios and patterns
2. **Reference databases essential** - compare to known samples
3. **Heat treatment affects some elements** - Ti mobility in sapphire
4. **Overlapping profiles exist** - some origins cannot be separated
5. **Combine with inclusions** - chemistry alone insufficient
6. **New deposits continually emerge** - databases need updates
