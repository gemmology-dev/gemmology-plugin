# Chemical Properties of Gemstones

Guidance on understanding the chemical composition, crystal chemistry, and colour-causing mechanisms in coloured gemstones.

## Chemical Classification

### Major Gem Mineral Groups

| Group | Chemical Type | Examples |
|-------|--------------|----------|
| Oxides | Metal + Oxygen | Corundum (Al₂O₃), Spinel (MgAl₂O₄), Chrysoberyl (BeAl₂O₄) |
| Silicates | Silicon + Oxygen based | Beryl, Tourmaline, Garnet, Quartz, Feldspar, Topaz |
| Phosphates | Phosphorus based | Apatite, Turquoise |
| Carbonates | Carbonate ion | Rhodochrosite, Malachite |
| Organic | Carbon-based life origin | Amber, Pearl, Coral |

### Silicate Sub-groups

| Sub-group | Structure | Examples |
|-----------|-----------|----------|
| Nesosilicates | Isolated tetrahedra | Garnet, Peridot, Zircon, Topaz |
| Sorosilicates | Double tetrahedra | Tanzanite (zoisite), Epidote |
| Cyclosilicates | Ring structures | Beryl, Tourmaline |
| Inosilicates | Chain structures | Spodumene, Diopside, Jadeite |
| Phyllosilicates | Sheet structures | Serpentine |
| Tectosilicates | 3D framework | Quartz, Feldspar |

## Colour-Causing Mechanisms

### Chromophores (Colour-Causing Elements)

| Element | Colours Produced | Examples |
|---------|------------------|----------|
| **Chromium (Cr³⁺)** | Red, Green | Ruby (red), Emerald (green), Chrome tourmaline |
| **Iron (Fe²⁺)** | Blue, Green | Aquamarine, Peridot |
| **Iron (Fe³⁺)** | Yellow, Brown | Citrine, some sapphires |
| **Iron (Fe²⁺/Fe³⁺)** | Blue | Blue sapphire (intervalence) |
| **Titanium (Ti⁴⁺)** | Blue | Blue sapphire (with Fe) |
| **Vanadium (V³⁺)** | Green, Purple | Colour-change gems, Tsavorite |
| **Manganese (Mn²⁺)** | Pink, Orange | Morganite, Spessartine, Rhodochrosite |
| **Manganese (Mn³⁺)** | Red, Purple | Rubellite (some) |
| **Copper (Cu²⁺)** | Blue, Green | Turquoise, Paraiba tourmaline |

### Colour Mechanisms

**1. Idiochromatic (Self-coloured)**
Colour-causing element is essential to mineral's formula.
- Peridot: Iron is part of formula (Mg,Fe)₂SiO₄
- Malachite: Copper is essential Cu₂CO₃(OH)₂
- Rhodochrosite: Manganese is essential MnCO₃

**2. Allochromatic (Other-coloured)**
Colour from trace impurities not essential to formula.
- Ruby: Cr³⁺ in Al₂O₃ (chromium replaces aluminium)
- Emerald: Cr³⁺ and/or V³⁺ in beryl
- Blue sapphire: Fe²⁺/Ti⁴⁺ intervalence charge transfer

**3. Colour Centres**
Defects in crystal structure cause colour.
- Amethyst: Irradiation-induced Fe colour centre
- Smoky quartz: Irradiation-induced Al colour centre
- Blue/yellow/pink diamonds (not covered)

### Blue Sapphire Colour Chemistry

The blue colour in sapphire is caused by **intervalence charge transfer** (IVCT):
- Fe²⁺ and Ti⁴⁺ ions occupy adjacent sites
- Electron transfers between them
- Absorbs orange-red light, transmits blue
- Formula: Fe²⁺ + Ti⁴⁺ ⇌ Fe³⁺ + Ti³⁺

### Ruby and Emerald: Same Chromophore, Different Colours

Both coloured by Cr³⁺, but:
- **Ruby (Al₂O₃)**: Cr³⁺ in octahedral site → red
- **Emerald (Be₃Al₂Si₆O₁₈)**: Cr³⁺ in different crystal field → green

The crystal field (surrounding atoms) determines which wavelengths are absorbed.

## Element Substitution (Solid Solution)

### Isomorphous Replacement

Elements can substitute for each other in crystal structures:

| Original | Substitute | Example |
|----------|------------|---------|
| Al³⁺ | Cr³⁺ | Ruby, Emerald |
| Al³⁺ | Fe³⁺ | Yellow sapphire |
| Mg²⁺ | Fe²⁺ | Peridot, Spinel |
| Ca²⁺ | Mn²⁺ | Spessartine |

### Garnet Solid Solutions

Garnets commonly show mixing between end-members:
- **Rhodolite** = Pyrope + Almandine (Mg-Fe series)
- **Malaya** = Pyrope + Spessartine (Mg-Mn series)
- **Mali** = Grossular + Andradite (Al-Fe series)

### Beryl Varieties by Trace Elements

All beryl: Be₃Al₂Si₆O₁₈ plus trace elements:

| Variety | Colour Agent | Colour |
|---------|--------------|--------|
| Emerald | Cr³⁺, V³⁺ | Green |
| Aquamarine | Fe²⁺ | Blue |
| Morganite | Mn²⁺ | Pink |
| Heliodor | Fe³⁺ | Yellow |
| Red beryl | Mn³⁺ | Red |
| Goshenite | None | Colourless |

## Trace Element Analysis

### Geographic Origin Indicators

Trace element signatures can indicate origin:

**Ruby**:
- Burmese: High Cr, low Fe, low Ti
- Thai: High Fe, low Cr
- Mozambique: Moderate Fe

**Emerald**:
- Colombian: Low Fe, Cr-dominant
- Zambian: High Fe, V-important
- Brazilian: Variable

**Blue Sapphire**:
- Kashmir: Low Fe/Ti ratio
- Ceylon: Moderate Fe/Ti
- Australian: High Fe/Ti

### Detection Methods

- **LA-ICP-MS**: Laser ablation - most accurate
- **EDXRF**: X-ray fluorescence - non-destructive
- **LIBS**: Laser-induced breakdown spectroscopy
